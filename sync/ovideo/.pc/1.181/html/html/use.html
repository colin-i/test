Using the program

Open the media edit options.
Use the top edit bar (navigator) to enter an uri and press Enter or the play button.
Save the navigator stream with the save button.
Look for informations about a stream with the info button.
Stop the play or the capture with the close button.
Use the search button to search for streams on an uri.
Set all the search options with the search preferences button.
Mux from two uri-s sources feature added, one uri is for video and one for audio.
View this file by pressing the help button.
The Search Preferences window:
  - the uri message will have the Body(content) part parsed.
  - the items will be extracted from the Start-> untill the <-End.
  - Wrap the items between two strings to create the final URI or leave blank.
The Stage mini-bar window:
  - The new media button is for adding a blank media.
  - The new media uri adds frames from the top uri bar location.
  - The preview button is for previewing the stage.
  - The pause button stops the preview.
  - The close button is for removing the bar.
  - The save to a file is for saving all the stage.
  - Expand a file with the stage data with this function.
  - The stage preferences is the next button.
  - Open mkv saved files on the stage.
  - Add sound files for mkv stage.
  - Capture video from the screen selecting a rectangle.
  - Add a frame with a selected width, height and color.
  - Add an image from a file button.
  - Remove the selected frame.
  - The frame length button is for increase or decrease a frame duration.
  - Split the selection into two parts.
  - Set the frame length to a number until they are equal; last can be truncated. Example: 8,1,1,1,1 and the number is 4 set on 8; becomes 4,4,2,1,1.
  - Frame panel for image tools.
   - The save button is for saving the selection to a file.
   - Add an image scaled and centered on the selected frame.
   - Create an image selecting width,height and color, then place over it the selected frame scaled and centered.
   - The scale button is for scaling an image to a new width and height.
   - Crop the selection.
   - Pencil tool for drawing.
   - Screen snapshot button.
   - Brighten or darken the selection.
   - Add an image over selection.
   - Rotate the image at left 90 degrees.
   - Rotate the image at right 90 degrees.
   - Add/Remove rows or columns at sides at all stage images.
  - The fade in/fade out feature is for selecting a frame and make an animation to a selected color.
  - The move in/move out is a reposition animation for the selected frame.
  - Uncover/Cover effects panel open button.
   - Uncover/Cover by side effect for the selected frame.
   - Uncover/Cover by center axis effect for the selected frame.
   - Uncover/Cover by rectangle shape effect for the selected frame.
   - Uncover/Cover by diamond shape effect for the selected frame.
   - Uncover/Cover by curved shape effect for the selected frame.
   - Uncover/Cover by diagonal effect for the selected frame.
  - Scale In/Scale Out animation.

To disable the update notification at startup there is an option at the stage preferences.
